# Code of Conduct

## Expected behavior

Participants in this project are expected to:

- be respectful and direct
- assume good intent
- give technical feedback without personal attacks
- keep discussion focused on the work
- help make the project welcoming to contributors with different backgrounds and experience levels

## Unacceptable behavior

The following behavior is not acceptable:

- harassment, threats, or intimidation
- discriminatory or demeaning language
- personal attacks or bad-faith arguments
- publishing private information without consent
- repeated disruption of issues, pull requests, or discussions

## Enforcement

If you experience or witness a serious issue, contact the repository maintainer privately through GitHub. Moderation actions may include comment removal, issue lock, rejection of contributions, or banning from the project space.
